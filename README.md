🧠 Therapy AI – Emotion-Aware Conversational Support
------------------------------------------------------
Therapy AI is a prototype web application that explores how emotion-aware language models can generate empathetic conversational responses. The system attempts 
to understand the emotional state of a user's message and produce a supportive reply with an appropriate tone.

The project focuses on system design, emotional reasoning, and conversational structure, rather than building a full mental health solution. 
It is intended as an experimental prototype and learning project, not a replacement for professional mental health care

💪🏻Motivation
-------------------------------------------------------
Many AI chatbots produce responses that are technically correct but emotionally flat. In conversations related to stress, loneliness, or anxiety, the tone of a response matters just as much as the content.
This project was built to explore whether a simple emotion-aware pipeline could improve conversational support by making responses feel more empathetic, reflective, and human-like.
Rather than generating responses blindly, the system first attempts to understand how the user might be feeling, and then adjusts the reply accordingly

⚙️ How It Works
-------------------------------------------------------
The system processes a user message through several stages:

User Message
The user writes a message describing how they feel or what they are experiencing.

Emotion Analysis
The system analyzes the text using a language model to identify emotional signals such as:

primary emotion
emotional intensity
positive / neutral / negative sentiment

Response Emotion Selection
Based on the detected emotion, the system determines the appropriate tone for the reply (supportive, calm, warm, etc.).

Therapeutic Response Generation
A second prompt generates a short therapy-style response that:

acknowledges the user’s feelings
provides gentle support
asks a reflective question

Voice Tone Tagging

The response also includes a tone label (for example: calm, supportive, warm).
This is designed for future integration with emotion-aware text-to-speech systems.
